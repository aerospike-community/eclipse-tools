Data Manipulation Language (DML)

INSERT INTO namespace[.set] (PK, bin names) VALUES (key, bin values)
DELETE FROM namespace[.set] WHERE PK = key
  • namespace is the namespace for the record.
  • set is the set name for the record.
  • key is the record's primary key.
  • bin names is a comma-separated list of bin names.
  • bin values is comma-separated list of bin values.
Examples:

INSERT INTO test.demo (PK, foo, bar) VALUES ('key1', 123, 'abc')
DELETE FROM test.demo WHERE PK = 'key1'
