Data Definition Language (DDL)

CREATE INDEX index ON namespace[.set] (bin name) NUMERIC|STRING
DROP INDEX namespace[.set] index
  • namespace is the namespace for the index.
  • set is the set name for the index.
  • index is the name of the index.
  • bin name is the name of the Bin to index
Examples:

CREATE INDEX idx_foo ON test.demo (foo) NUMERIC
DROP INDEX test.demo idx_foo
